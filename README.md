**如何查询linux下各种库或者应用的版本号**

一般而言，都是输入以下命令 程序名 -v 或者 程序名 -V 或者 程序名 version

具体的可以输入 程序名 -h 查看帮助文档，里面会写很详细的有关于这个库或者程序的一些命令提示符
